TigerOS
====================


## Instructions
### Setup 
* Clone this repo
```  
  git clone https://github.com/RITlug/TigerOS.git
```
### Build the ISO
From the directory where you cloned this repo run the commands below. You may need to preface it with `sudo` if you do not have root privileges.
```
chmod +x make-iso.sh
bash make-iso.sh
```
